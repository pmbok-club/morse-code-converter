# morse_code_converter
アルファベットを01のモールス信号に変換するスクリプト

2019年静大祭に出展した脱出ゲーム「ヤバい研究室への潜入」ポスターの背景に使用した。

## 使用例

```
$ python morse_code_converter.py "yabaken"                               
 1011 01 1000 01 101 0 10
```
